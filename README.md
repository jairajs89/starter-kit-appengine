# Welcome to your AppEngine starter kit

### Setup

[Install Python SDK for Google AppEngine](https://cloud.google.com/appengine/docs/standard/python/download)

### Development

```sh
make debug
# your debug server is now running at localhost:8080
```

Run tests:

```sh
make test
```

### Deployment

```sh
make deploy
```
